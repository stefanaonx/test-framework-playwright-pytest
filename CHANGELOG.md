# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## 0.0.1 - 2025-11-24

### Added

*   Initial project setup for Kiwi.com flight search automation.
*   Implemented Page Object Model (POM) for the homepage.
*   Defined Gherkin feature file for one-way flight search scenario.
*   Implemented step definitions using Playwright and Pytest-BDD.
*   Added `requirements.txt` for dependency management.
*   Included `Dockerfile` for Docker containerization.
*   Configured GitHub Actions for CI/CD.
*   Created `README.md` with project overview, setup, and usage instructions.
